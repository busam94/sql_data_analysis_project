
# SQL Layoffs Data Cleaning & EDA Project

# Overview
This project demonstrates data cleaning and exploratory data analysis (EDA) using SQL on a dataset of employee layoffs. 

# Dataset
- Source: Layoffs dataset (employee records, departments, roles, dates)
- Format: CSV exported from SQL

# Tools Used
- SQL (MySQL)
- Optional: Excel / Power BI for future visualization

# Folder Structure
- data/     → CSVs of layoffs dataset
- code/     → SQL scripts for cleaning and EDA
- README.md → Project overview and instructions

# How to Use
1. Open CSV files in `data/`
2. Run SQL scripts in `code/` to reproduce cleaning and EDA
3. Outputs (tables and summaries) will appear in your SQL client
